# Leaderboard Website

A simple, static leaderboard website built with HTML and Bulma CSS framework. Easily deployable via GitHub Pages.

## Project Structure

- `website/index.html` — Main leaderboard page
- `website/assets/style.css` — Custom styles
- `readme.md` — Project documentation

## How to Use

1. Edit `website/index.html` to update leaderboard entries.
2. Push changes to your GitHub repository.
3. Enable GitHub Pages in the repository settings, selecting the `website` folder as the source.

## Customization
- To change the CSS framework, update the `<link>` in `index.html`.
- For more features, add more HTML or custom CSS as needed.
